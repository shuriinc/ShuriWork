Tags

Tags are the way to categorize things to impart useful meaning. Tags may be logically grouped together via a user type to create different "sets" of tags. Users may create any number of tags, but each must belong to one usertype or the generic public usertype "tags".

Resource Description
GET tags Returns an array of all user types of type tag that are available to the requestor. Tags are hydrated within usertype.
GET tags
?groups=[ group ids ]
&usertypes=[ user type ids ]
&nameBegins=[ string ]
Returns an array of all user types of type tag that match the filters and that are available to the requestor. Tags are hydrated within usertype.
GET tag?id=[ id ] Returns a fully hydrated tag.
POST tags
?subscriptionId=[ group id ]
Create a new tag in the collection requested or the user's private collection by default. If the id is empty a create occurs, else an update to the existing record.
DELETE tag Deletes an existing tag. The tag and all of it's relations are permanently deleted. User must own the tag or be a member of the owning group.
